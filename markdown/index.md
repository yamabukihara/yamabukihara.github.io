# Markdown常用语法

## 一、标题
使用#符号可以表示1-6级标题，一级标题对应一个#符号，二级标题对应两个#符号，以此类推
```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

```
#### 显示效果：
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

---

## 二、段落
Markdown段落没有特殊格式，直接编写文字即可，   
段落换行号是使用两个以上的空格加回车。
### 2.1 字体
- 用1个*或_表示斜体
- 用2个*或_表示粗体
- 用3个*或_表示粗斜体
```markdown
*斜体文字*
_斜体文字_
**斜体文字**
__斜体文字__
***斜体文字***
___斜体文字___
```
#### 显示效果：
*斜体文字*  

_斜体文字_  

**斜体文字**  

__斜体文字__  

***斜体文字***  

___斜体文字___  

---

### 2.2 分割线
可以在一行中用三个以上的星号*、减号-、底线_来建立一个分隔线，行内不能有其他东西，你也可以在星号或减号蹭插入空格。
```markdown
***
* * *
******
- - -
------
```
#### 显示效果：
***
* * *
******
- - -
------

### 2.3 删除线
如果段落上的文字要添加删除线，只需要在文字的两端加上两个波浪线~~即可
```markdown
baidu.com
sina.com
~~tencent.com~~
```
#### 显示效果：
baidu.com
sina.com
~~tencent.com~~

### 2.4 下划线
下划线可以通过HTML的标签来实现
```markdown
<u>带下划线文本</u>
```
#### 显示效果：
<u>带下划线文本</u>

### 2.5 脚注
脚注是对文本的补充说明。
```markdown
这是一段文字，需要加脚注[^1]。
这是另一段文字，有另一个脚注[^2]。
[^1]: 这是第一个脚注的内容，可以很长，支持 **加粗**、`代码`、链接等 Markdown 语法。
[^2]: 这是第二个脚注的内容。
```
#### 显示效果：
这是一段文字，需要加脚注[^1]。  

这是另一段文字，有另一个脚注[^2]。  

[^1]: 这是第一个脚注的内容，可以很长，支持 **加粗**、`代码`、链接等 Markdown 语法。  
[^2]: 这是第二个脚注

## 三、列表

### 3.1 有序列表与无序列表
无序列表使用星号(*)、加号(+)或者减号(-)进行标记
```markdown
* 第一项
* 第二项
* 第三项

+ 第一项
+ 第二项
+ 第三项

- 第一项
- 第二项
- 第三项
```
#### 显示效果：
* 第一项
* 第二项
* 第三项

+ 第一项
+ 第二项
+ 第三项

- 第一项
- 第二项
- 第三项

有序列表直接在文字前加上1. 2. 3.表示，符号和文字之间需要加上一个空格
```markdown
1. 第一项
2. 第二项
3. 第三项
```
#### 显示效果：
1. 第一项
2. 第二项
3. 第三项

### 3.2 列表嵌套
列表嵌套只需在子列表的选项前添加四个空格即可
```markdown
1. 第一项：
    - 第一项嵌套的第一个元素
    - 第一项嵌套的第二个元素
2. 第二项：
    - 第二项嵌套的第一个元素
    - 第二项嵌套的第二个元素

```
#### 显示效果：
1. 第一项：
    - 第一项嵌套的第一个元素
    - 第一项嵌套的第二个元素
2. 第二项：
    - 第二项嵌套的第一个元素
    - 第二项嵌套的第二个元素

## 四、区块
Markdown区块引用是在段落开头使用>符号，然后后面紧跟一个空格符号
```markdown
> 区块引用
> Markdown教程  
> 学的不仅是技术更是梦想
```
#### 显示效果：
> 区块引用
> Markdown教程  
> 学的不仅是技术更是梦想

另外区块是可以嵌套的，一个>符号是最外层，两个符号是第一层嵌套，以此类推
```markdown
> 最外层
>> 第一层嵌套
>>> 第二层嵌套
```
#### 显示效果：
> 最外层
>> 第一层嵌套
>>> 第二层嵌套

### 4.1 区块中使用列表
```markdown
> 区块中使用列表
> 1. 第一项
> 2. 第二项
> + 第一项
> + 第二项
> + 第三项
```
#### 显示效果：
> 区块中使用列表
> 1. 第一项
> 2. 第二项
> + 第一项
> + 第二项
> + 第三项

### 4.2 列表中使用区块
```markdown
* 第一项
    > Markdown教程
    > 学的不仅是技术更是梦想
* 第二项
```
#### 显示效果：
* 第一项
    > Markdown教程  
    > 学的不仅是技术更是梦想
* 第二项

## 八、表格
Markdown制作表格使用|来分隔不同的单元格，使用-来分隔表头和其他行。

```markdown
|表头1|表头2|
|----|----|
|单元格11|单元格12|
|单元格21|单元格22|
```
#### 显示效果：
|表头1|表头2|
|----|----|
|单元格11|单元格12|
|单元格21|单元格22|

### 8.1 对齐方式

可以设置表头的对齐方式
- -: 设置内容或标题栏右对齐
- :- 设置内容或标题栏左对齐
- :-: 设置内容或标题栏居中对齐

```markdown
|左对齐|居中对齐|右对齐|
|:----|:----:|----:|
|单元格11|单元格12|单元格13|
|单元格21|单元格22|单元格23|
```
#### 显示效果：
|左对齐|居中对齐|右对齐|
|:----|:----:|----:|
|单元格11|单元格12|单元格13|
|单元格21|单元格22|单元格23|

## 九、高级技巧

### 9.1 转义
Markdown使用了很多特殊符号来表示特定的意义，如果需要显示特定的符号则需要使用转义字符，Markdown使用反斜杠转义特殊字符  
```markdown
**文本加粗**
\*\*正常显示星号\*\*
```
#### 显示效果：
**文本加粗**
\*\*正常显示星号\*\*

#### 支持转义的符号
```markdown
\   反斜线
`   反引号
*   星号
_   下划线
{}  花括号
[]  方括号
()  小括号
#   井字号
+   加号
-   减号
.   英文句点
!   感叹号
```

### 9.2 数学公式
当你需要在编辑器中插入数学公式时，可以使用两个美元符 $$ 包裹 TeX 或 LaTeX 格式的数学公式来实现。
```markdown
$$
\int_0^1 {x^2} \,{\rm d}x
$$
```
#### 显示效果：
$$
\int_0^1 {x^2} \,{\rm d}x
$$
