# 程序设计

### 核心概念
+ 变量、作用域、生命周期
+ 函数式、命令式、面向对象三种范式
+ 内存管理（栈、堆、手动/自动）
+ 所有权与借用
+ 并发模型（线程、异步、actor）
+ 类型系统（静态VS动态，强类型VS弱类型）
+ 错误处理（异常 VS Result ）
+ 元编程与宏

### 编程语言

##### C++
- 堆、栈、虚表、RAII、UB（未定义行为）
- 模板元编程（能写出Turing Complete的代码）
- 手动管理内存

##### Rust
- 所有权、借用、生命周期
- trait比传统继承更优雅
- async/await的Tokio生态
- 学会Fearless Concurrency

##### JavaScript
- 原型继承 vs 类继承
- 事件循环与异步（Promise -> async/await）
- 动态类型的灵活与隐患（TS补上类型安全）
- 函数是一等公民
